# Georgia Elections 2026 | Financial Data Scraper
A specialized web automation and data engineering tool designed to aggregate campaign finance data from state-level government portals for public transparency.

## Tech Stack
* **Language:** Python
* **Automation:** Playwright / BeautifulSoup4
* **Data Processing:** Pandas
* **Environment:** JavaScript (for dynamic content handling)

* [cite_start]**Dynamic Scraper Logic:** Engineered to navigate and extract data from complex, JavaScript-rendered government portals[cite: 512].
* [cite_start]**Automated Data Pipeline:** Built pipelines to clean and structure raw HTML/JSON data into analysis-ready CSV/JSON formats[cite: 513].
* [cite_start]**Robust Error Handling:** Implemented retry logic and exception handling to maintain data integrity across high-volume scraping tasks despite varying network conditions[cite: 514].

## Installation & Usage
1. Clone the repo: `git clone https://github.com/CJohnson0228/ga-elections-2026.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the scraper: `python main.py`
